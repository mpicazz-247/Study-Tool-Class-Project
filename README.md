# Study-Tool-Class-Project

The Study tool is a JavaFX application designed to help students manage their time and tasks effectively by implementing a Pomodoro timer and To-Do list into one functional computer program. 

Features: 
Pomodoro Timer with 25/5 and 50/10 study/break modes. 
Start/Stop, reset, and skip timer functions
Dropdown menu for selecting desired amount of repetitions (up to 8)
To-Do List with: 
  Add, Delete, and name task functions 
  Toggle button for task completion

Technology Stack:
Java and JavaFX
Scene Builder 
IntelliJ IDEA
GitHub

Installation: 
clone the repository 
open the project in your preferred Java IDE (IntelliJ IDEA recommended)
run the application from Main.java
